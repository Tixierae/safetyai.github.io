JANUARY 19, 2026 19 JANVIER 2026

Custom Database & System Updates Mises à jour Base de Données & Système

We have deployed an update focusing on the reliability of the Custom Database retrieval system, alongside enhancements to the Regulations mode and a new self-healing link mechanism. Nous avons déployé une mise à jour axée sur la fiabilité du système de recherche de la Base de Données Personnalisée, ainsi que des améliorations du mode Réglementations et un nouveau mécanisme d'auto-réparation des liens.

Smarter Search Calls Appels de Recherche Plus Intelligents

ChatSafetyAI is now aware of the file list within the selected custom database. This context allows the AI to generate more accurate search queries, resulting in better answers. ChatSafetyAI connaît désormais la liste des fichiers au sein de la base de données sélectionnée. Ce contexte permet à l'IA de générer des requêtes de recherche plus précises, améliorant ainsi la qualité des réponses.

Systematic Search Triggering Déclenchement Systématique de la Recherche

We fixed an issue where the custom database mode would sometimes fail to trigger more than once in the same conversation. Now, whenever a custom database is selected, a search call is performed systematically for every user query. Nous avons corrigé un problème où le mode base de données personnalisée ne se déclenchait pas plusieurs fois de suite dans une conversation. Désormais, lorsqu'une base de données est sélectionnée, une recherche est effectuée systématiquement.

Index Update Synchronisation de l'Index

Files deleted from a custom database were not removed from the search index, which could lead to ChatSafetyAI citing documents that no longer existed, with broken URLs. This has been fixed: the index now syncs instantly upon file deletion. Les fichiers supprimés d'une base de données personnalisée n'étaient pas retirés de l'index de recherche, ce qui pouvait amener ChatSafetyAI à citer des documents inexistants, avec des liens invalides. Ce problème est corrigé : l'index se synchronise désormais instantanément lors de la suppression de fichiers.

Support for Accented Filenames Support des Noms de Fichiers Accentués

The indexing engine now supports folder and file names containing accented characters (e.g., "Sécurité", "Procédure"). Le moteur d'indexation prend désormais entièrement en charge les noms de dossiers et de fichiers contenant des caractères accentués (ex : "Sécurité", "Procédure").

Regulations Mode Enhancements Améliorations du Mode Réglementations

We improved the system instructions to better handle multiple regulation calls within the same conversation. We also updated the allowlist of authoritative domains for Quebec to ensure better results. Nous avons amélioré les instructions système pour mieux gérer les appels réglementaires multiples au sein d'une même conversation. Nous avons également mis à jour la liste des domaines autorisés pour le Québec afin d'assurer de meilleurs résultats.

Automatic URL Verification Vérification Automatique des URL

We introduced a self-healing mechanism for citations. If the model detects a broken or invalid URL in its sources, it now triggers an automatic web search to find and propose valid alternative links. Nous avons introduit un mécanisme d'auto-correction pour les citations. Si le modèle détecte une URL brisée ou invalide dans ses sources, il déclenche désormais une recherche web automatique pour trouver et proposer des liens alternatifs valides.
