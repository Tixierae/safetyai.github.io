Company-Wide Databases & Fixes Bases de Données d'Entreprise & Corrections

Hi there! 👋 Here’s a new feature we thought you’d like. Bonjour ! 👋 Voici une nouvelle fonctionnalité qui devrait vous plaire.

Company-Wide Databases ✨🏢 Bases de Données à l'Échelle de l'Entreprise ✨🏢

The admin users of each organization can now create and populate databases visible to anyone in the organization! Regular users can view the files, but not add or remove files or create/delete company-wide databases. Of course, every user still manages their own personal databases, as before. Les administrateurs de chaque organisation peuvent désormais créer et alimenter des bases de données visibles par tous les membres de l'organisation ! Les utilisateurs réguliers peuvent consulter les fichiers, mais ne peuvent ni ajouter ni supprimer de fichiers, ni créer ou supprimer des bases de données d'entreprise. Bien entendu, chaque utilisateur gère toujours ses propres bases de données personnelles, comme auparavant.

👉 Note: The admin users for your company are those who can access the ChatSafetyAI Usage Dashboard. Contact us to add or remove people. 👉 Note : Les administrateurs de votre entreprise sont ceux qui ont accès au tableau de bord d'utilisation de ChatSafetyAI. Contactez-nous pour ajouter ou retirer des personnes.

💡 Remember: To let ChatSafetyAI ground its responses in a given database, select that database in the list above your query before submitting it! 💡 N'oubliez pas : Pour que ChatSafetyAI fonde ses réponses sur une base de données spécifique, sélectionnez-la dans la liste au-dessus de votre requête avant de l'envoyer !

⚠️ For follow-up queries that do not require a database search (such as document generation, HECA, predictions, etc.), clear your selection! ⚠️ Pour les requêtes qui ne nécessitent pas de recherche dans la base de données (comme la génération de documents, HECA, les prédictions, etc.), effacez votre sélection !

Chat Sharing, Parallel Session Fixed 🐞🛠️ Partage de Chat, Session Parallèle Corrigée 🐞🛠️

The chat sharing functionality works again. The "Parallel Session" permanent hold has been addressed. If you face that issue again, you should now be able to start a new session after 60 secs at most. Thanks everyone for the bug reports. La fonctionnalité de partage de chat fonctionne à nouveau. Le blocage permanent lié aux « Sessions Parallèles » a été résolu. Si vous rencontrez à nouveau ce problème, vous devriez maintenant pouvoir démarrer une nouvelle session après 60 secondes maximum. Merci à tous pour vos signalements de bugs.

What's Next? 😍 La Suite ? 😍

We’re working on user preferences, memories, chat renaming, chat search engine, advanced spreadsheet reasoning and visualization, image generation, covering new regulation jurisdictions, and more! Submit your suggestions here. Nous travaillons sur les préférences utilisateur, la mémoire, le renommage des chats, un moteur de recherche de conversations, le raisonnement et la visualisation avancés sur les feuilles de calcul, la génération d'images, la couverture de nouvelles juridictions réglementaires, et plus encore ! Soumettez vos suggestions ici.
